Inception

Visão do Projeto​

O EduPro é um sistema de gestão e avaliação de unidades curriculares. Com objetivo de ajudar a organização das cadeiras, geração e correção automática de testes e a criação de manuais de apoio.

Personas​

  • Gestor
  • Regente
  • Docente
  • Aluno (Indireta)

Gestor​

User Stories​

Como Gestor quero adicionar/remover cadeiras para organizar o curso e definir a oferta formativa.
Como Gestor quero atribuir/remover regentes a cadeiras para que atualizar a regência da cadeira.
Como Gestor quero atribuir/remover professores a cadeiras para que ajustar a equipa docente conforme necessário.

Use Cases​

  • Adicionar cadeiras
  • Remover cadeira
  • Atribuir regentes a cadeiras
  • Remover regentes de cadeiras
  • Atribuir docentes a cadeiras
  • Remover docentes de cadeiras
  • Criar o template de cada cadeira

Regente​

User Stories​

Como Regente quero adicionar/remover módulos para atualizar os objetivos da cadeira.
Como Regente quero adicionar/remover perguntas para ir de encontro com os objetivos da cadeira.
Como Regente quero atribuir/remover docentes à cadeira para manter a equipa de docentes atualizada.
Como Regente quero editar permissões dos docentes da cadeira para poder personalizar o que cada docente pode fazer.
Como Regente quero adicionar/remover alunos à cadeira para manter a lista de alunos atualizada.
Como Regente quero gerar testes para garantir que cada aluno tem um teste único.
Como Regente quero exportar notas de testes para ter uma lista organizada do aproveitamento de cada estudante.
Como Regente quero consultar estatísticas de desempenho dos alunos para entender as suas dificuldades.
Como Regente quero gerar manuais para garantir material de estudo aos meus alunos.

Extras​

Como Regente quero lançar notas de testes para que os alunos sejam notificados do seu aproveitamento por e-mail.

Use Cases​

  • Adicionar módulos
  • Remover módulos
  • Adicionar perguntas
  • Remover perguntas
  • Atribuir docentes à cadeira
  • Remover docentes da cadeira
  • Editar permissões dos docentes da cadeira
  • Adicionar alunos à cadeira
  • Remover alunos da cadeira
  • Gerar testes
  • Exportar notas de testes
  • Consultar estatísticas de desempenho dos alunos
  • Gerar manuais
Extras​
  • Lançar notas de testes

Docente​

User Stories​

Como Docente quero consultar cadeiras para ver se houve alterações no material da cadeira.
Como Docente quero consultar o banco de perguntas da cadeira para ver a que tipo de questões os alunos são sujeitos.
Como Docente quero associar alunos a testes para agilizar o processo de correção.

Use Cases​

  • Consultar cadeiras
  • Consultar banco de perguntas da cadeira
  • Associar alunos a testes

Aluno (Indireta)​

Descrição​

Utiliza os manuais e os resultados gerados pela aplicação.
